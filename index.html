ОБЩИЕ СВЕДЕНИЯ
Я обладаю 4х летним опытом работы системным администратором в учреждении с доменной сетью из 195 ПК. Работал с операционками Windows 7 / 10 / 11, Ubuntu и Windows Server 2016. В локальной сети были развёрныт 1С, медицинские информационные системы и разработанное мной ПО.

За время работы мной были разработаны и внедрены ряд программ для оптимизации работы:

- Epicrisis_packer (автоматический сбор, сортировка по номерам поликлиник и архивация для последующей отправки). Стек технологий: Python
- Deed_collector (раз в месяц "парсит" несколько электронных ящиков, на которые приходят акты выполненных работ от субподрядчиков и пересылка их в бухгалтерию). Стек технологий: Python, Celery, Docker
- Dokumentoz (система для формирования служебных записок, пересылки их исполнителям и отслеживания их статусов исполнения с напоминаниями через telegram-бот). Стек технологий: Python, Django REST Framework, Aiogram, Celery, TypeScript, React, PostgreSQL, Docker.
